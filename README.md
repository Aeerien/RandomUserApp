Задание 

Необходимо разработать приложение, отображающее список людей и растояния от них до пользователя. 

Экран поиска
1. Каждый элемент списка должен содержать аватар, имя и расстояние до пользователя.
2. Положение пользователя должно быть получено от GPS.
3. При нажатии на человека, информация о нём закрепляется над списком и не пропадает при скролле.
4. При повторном нажатии выбор человека отменяется.
5. Пока есть выбранный человек, расстояния остальных людей отображаются относительно выбранного.

Дополнительные требования
1. Все данные о людях и их координаты должны быть получены из сервиса. Других требований к сервису, списку его «запросов» и формату данных нет.
2. Каждые три секунды положение людей должно меняться со случайным смещением.
3. Для реализации можно использовать любые сторонние библиотеки.
4. К дизайну нет требований, кроме соблюдения HIG (Human Interface Guidelines) и здравого смысла.
5. Нужна поддержка как минимум двух последних версий iOS.
6. Проект должен собираться без дополнительных действий, кроме установки зависимостей (например, carthage bootstrap или pod install).
7. Проект должен собираться без ошибок и предупреждений на последней релизной версии Xcode.
8. Желательно использовать последнюю релизную версию Swift.
9. А также желательно использовать SwiftUI и Swift Concurrency.
